~~:INTRODUCTION :~~

DocmaQ (abbreviation to Document Maker) is a Credential Publishing Software which publishes/prints various Credentials of SNIST (Sreenidhi Institute of Science and Technology)

These Credentials include :

- Bonafide
- Conduct
- Transfer Certificate

DocmaQ provides the following features :

- User Authentication
- Database connectivity
- Manual Credential Publishing
- Credential Log Managing and Accountabality
- Various Preferences and Settings
- Quicker and Easier Navigation

DocmaQ is Exclusively Developed to meet the Requirements of SNIST(www.sreenidhi.edu.in)

---------------------------------------------------------------------------
~~: REQUIREMENTS :~~

- Personal computer with AMD or i386 or higher processor
- LAN (for remote Database Operation only)
- 100 MB Hard Disk space 
- 128 MB RAM
- Windows  Xp / Vista / 7

- MySQL Server 5.0 (or higher)
  - Database Tables :
    - student_info
    - student_academic
    - student_admission
    - teacher_info
    - login_info
    - tc_log
